# ScholarsHub – Education Platform with AI Chatbot

ScholarsHub is a full-stack web application that provides students with access to educational programs, application submission, and an AI-powered chatbot for assistance.

The platform combines modern UI + backend APIs + AI integration to deliver a seamless user experience.

## Features

### 🎯 Core Features

- 📚 View available programs  
- 📝 Apply for programs  
- 📩 Email confirmation after application  
- 🔐 Authentication system (login/signup)  
- 🛠️ Admin routes for management  

### 🤖 AI Chatbot

- Gemini-powered chatbot  
- Answers user queries  
- Real-time responses  
- Typing indicator for better UX  

---


## Tech Stack

- **Frontend:** React (Vite)  
- **Styling:** Custom CSS  
- **Backend:** Node.js, Express.js  
- **AI:** Google Gemini API (gemini-2.5-flash)  
- **Deployment:** Vercel 


## Application Flow

1. User visits the platform  
2. User can:
   - View programs  
   - Apply for a program  
   - Chat with AI assistant  
3. Backend processes request  
4. Database stores application data  
5. Email confirmation is sent  
6. AI chatbot responds dynamically

## Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/harshtiwari2022/devcoach-AI.git
   ```
2. Navigate to the project directory:
    ```bash
    cd ScholarsHub 
    ```
3. Install dependencies:
    ```bash 
    npm install
    ```
4. Start the development server:
    ```bash
    npm run dev
    ```
5. Start Run backend::
    ```bash
    node index.js
    ```
