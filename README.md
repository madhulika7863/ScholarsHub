ScholarsHub is a full-stack web application that provides students with access to educational programs, application submission, and an AI-powered chatbot for assistance.

The platform combines modern UI + backend APIs + AI integration to deliver a seamless user experience.

## Features

### 🎯 Core Features
👨‍🎓 User Side
- 📚 View available programs  
- 📝 Apply for programs  
- 📩 Contact Form Submission
- 🔐 Authentication system (login/signup)  
- 🛠️ Admin routes for management  
- 📱 Fully Responsive & Modern UI
### 🤖 AI Chatbot

- Gemini-powered chatbot  
- Answers user queries  
- Real-time responses  
- Typing indicator for better UX  

## 🛠️ Admin Side

- 👑 Protected Admin Dashboard
- 📊 View All Applications
- 📬 Manage Contact Messages
- ❌ Delete Records (Applications & Contacts)
---


## Tech Stack

- **Frontend:** React (Vite)  
- **Styling:** Custom CSS , Axios
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB, Mongoose
- **AI:** Google Gemini API (gemini-2.5-flash)  
- **Deployment:** Vercel, render


## Application Flow
1. User visits the platform  
2. User can:
   - View programs  
   - Apply for a program  
   - Chat with AI assistant  
3. Backend processes request  
4. Database stores application data   
5. AI chatbot responds dynamically

## Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/madhulika7863/ScholarsHub.git
   ```
2. Navigate to the project directory:
    ```bash
    cd ScholarsHub 
    ```
3. Install dependencies:
    ```bash 
    npm install
    ```
4. Start the development server:
    ```bash
    npm run dev
    ```
5. Start Run backend::
    ```bash
    node server.js
    ```
